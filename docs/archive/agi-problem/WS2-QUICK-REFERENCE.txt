WHAT EACH DOES
================================================================================

SOAR (REASONING LAYER):
  - Handles NOVEL problems
  - Explores alternatives under uncertainty
  - Learns STRATEGIES (not just facts)
  - Takes 5-15 seconds per problem
  - Provides transparent, explainable reasoning
  - Proves it works: Chess, robotics, debugging

ACT-R (LEARNING LAYER):
  - Handles FAMILIAR problems
  - Makes rapid decisions (200ms)
  - Learns from repeated feedback
  - Improves continuously (novice → expert)
  - Provides fast responses
  - Proves it works: Learning, skill development

BOTH TOGETHER:
  - Novel problems: Use SOAR (thorough reasoning)
  - Familiar problems: Use ACT-R (fast path)
  - Learning: Both mechanisms improve simultaneously
  - System improves over time (month 1 > month 2 > month 3)
